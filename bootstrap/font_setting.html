기본 boostrap.css를 사용하면 윈도우는 default font or 굴림체, 맥은 default font가 적용됨.

브라우저마다 다르게 나오는 한글을 같게 하는 방법은
1. 윈도우 비스타 default font인 '맑은 고딕'을 사용 - bootstrap.css 파일 수정
2. font-face 속성을 이용하여 다른 font를 사용 - style 설정

1. 윈도우 비스타 default font인 '맑은 고딕'을 사용 - bootstrap.css 파일 수정

1-1. bootstrap.css 파일 수정
body { font-family :"Helvetica Neue", Helvetica, Arial, "맑은 고딕", "Malgun gothic", sans-serif; }
1-2. style 추가 ==> 권장
<!-- Bootstrap -->
<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" rel="stylesheet">
<style>
    body{font-family: "Helvetica Neue", Helvetica, Arial, "맑은 고딕", "Malgun gothic", sans-serif;}
</style>

<!-- Bootstrap -->
<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" rel="stylesheet">
<link href="css/custom.css" rel="stylesheet">
==> 윈도우 xp 이하, 오피스 2010 미만 설치 PC에서는 보이지 않음.
1-3. Compile customized css on bootstrap site
https://getbootstrap.com/customize/#typo ==> 별로 사용되는 방법은 아님. 개인화된 bootstrap css 파일을 만드는 것임.

2. font-face 속성을 이용하여 다른 font를 사용 - style 설정

2-1. 직접 font file 설치
1) custom.css 파일 작성
@font-face {
                        font-family: ‘NanumGothic’
                        src:url("NanumGothic.eot");
                        src:local(""), url("NanumGothic.woff") format("woff");
                    }
body {font‐family: "Helvetica Neue", Helvetica, Arial, "NanumGothic", sans‐serif;}
2) custom.css 파일 적용
<link href="css/bootstrap.min.css" rel="stylesheet">
                    <link href="css/custom.css" rel="stylesheet">
2-2. Google font 이용
1) css 파일에 폰트 주소 추가
@import url(https://fonts.gooleapis.com/earlyaccess/nanumgothic.css)
참고 사이트 : https://fonts.google.com/
<link> tag를 사용하면 반드시 HTML 문서의 <head> tag 안에 넣어줘야 하지만, @import를 사용하게 되면 HTML의 <head> tag나 별도의 css 파일에 넣어서 사용할 수 있다. 관리적인 측면에서는 편의를 위해서 @import를 사용하면 HTML 문서가 깔끔해진다.자바스크립트를 이용해서 HTML문서의 <head> tag 안에 넣어줄 수도 있다.
2) 사용하려는 폰트를 적용하고자 하는 class에 attribute 추가
body {font‐family: ‘Nanum Gothic’, sans-serif;}
!주의 윈도우에서 사용되는 굴림이나 돋움체는 비트맵용 글꼴로 제작된 서체이기 때문에 윈도우에서는 12px 또는 11px일 때가 가장 가독성이 좋다. 14px 이상일 때는 글꼴이 깨지는 현상이 발생하기도 하니, 14px이상의 글꼴을 사용해야 한다면 나눔고딕이나 맑은고딕을 사용하자!
